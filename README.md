# AttoGCS

AttoGCS는 Mission Planner의 일반사용자의 사용의 복잡성을 줄이기 위하여 시작하였습니다.

Mission Planner는 드론 개발자에게는 다양한 테스트를 할수 있도록 되어 있어 개발자에게는 다양한 테스트를 할수 있도록 되어있습니다.

하지만 일반 사용자들에게 있어서는 필요 없는 부분이 너무 많아서 사용하기에 불편한 점이 많습니다. 

AttoGCS는 일반 사용자를 위하여 불필요한 부분을 줄이고 단순한 기능으로 만들고자 합니다.

또한 특정 제품군으로 한정지어서 편의성을 높이고자 합니다.

라이센스는 Mission Planner와 동일합니다.
